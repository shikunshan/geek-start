# GeekStart 极客终端导航页 产品需求文档

## 1. 产品概述

GeekStart 是一个外观与交互完全模仿 Linux 终端的浏览器导航起始页，用户通过输入命令完成搜索、书签管理、背景切换、音乐播放、系统状态查看等操作，主打极客美学与高效率。

- **核心目标**：打造全黑命令行风格的浏览器主页，通过「输入命令 → 实时响应」的方式完成所有操作
- **目标用户**：开发者、极客、喜欢命令行操作的效率用户
- **产品价值**：高度可定制、离线可用、零后端的纯前端方案，提供沉浸式终端体验

---

## 2. 核心功能

### 2.1 功能模块

1. **终端核心**：输入输出、光标闪烁、命令历史、Tab 补全
2. **命令系统**：命令注册与分发、help / clear / history 内置命令
3. **背景管理**：切换全屏背景图片、随机轮播、纯黑模式
4. **书签系统**：显示、添加、删除书签，编号快速跳转
5. **搜索功能**：多搜索引擎支持（Google / GitHub / npm / DuckDuckGo / arXiv）
6. **音乐/白噪音**：背景音乐播放、白噪音（雨声/海浪/咖啡馆）
7. **系统状态**：neofetch、uptime、whoami、top 模拟
8. **主题系统**：多种配色主题（default / nord / dracula / gruvbox / monokai）
9. **配置系统**：用户配置持久化、默认搜索引擎、用户名等
10. **彩蛋命令**：sudo rm -rf /、fortune、sl、yes、starwars、cowsay

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 终端主页 | 输出区域 | 显示命令历史和执行结果，支持自动滚动 |
| 终端主页 | 输入行 | `>` 提示符 + 可编辑输入区 + 闪烁光标 |
| 终端主页 | 背景层 | 全屏背景图片/纯色，支持淡入淡出切换 |
| 终端主页 | 音频层 | 背景音乐/白噪音播放控制 |

## 3. 核心流程

### 3.1 用户操作流程

用户打开页面 → 终端初始化（显示欢迎信息）→ 用户输入命令 → 按 Enter 执行 → 输出结果 → 等待下一条命令

```mermaid
flowchart LR
    A["打开页面"] --> B["终端初始化"]
    B --> C["显示欢迎信息 + 提示符"]
    C --> D["用户输入命令"]
    D --> E["按 Enter 执行"]
    E --> F["命令解析与匹配"]
    F --> G["执行命令 handler"]
    G --> H["输出结果到终端"]
    H --> I["显示新提示符"]
    I --> D
```

### 3.2 关键交互

- **↑ / ↓ 键**：浏览历史命令
- **Tab 键**：自动补全命令名
- **Ctrl + L**：清屏
- **Ctrl + C**：取消当前执行

## 4. 用户界面设计

### 4.1 设计风格

- **设计方向**：复古终端风格 + 极客美学
- **主色调**：黑色背景 (#000000) + 绿色文字 (#00ff00) 经典终端配色
- **辅助色**：红色（错误）、黄色（警告）、蓝色（信息）、青色（高亮）
- **字体**：等宽字体（JetBrains Mono / Fira Code / Consolas）
- **视觉效果**：光标闪烁、打字回显、CRT 扫描线效果、微噪点纹理

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 终端主页 | 输出区域 | 等宽字体、文字左对齐、行高 1.5、支持 ANSI 颜色 |
| 终端主页 | 输入行 | `>` 提示符、闪烁光标（▌）、输入无框无边 |
| 终端主页 | 背景层 | 全屏覆盖、低透明度遮罩、模糊效果、淡入淡出过渡 |

### 4.3 响应式

- 桌面端优先设计
- 全屏终端，padding 约 2rem
- 最小宽度支持 800px

### 4.4 动效设计

- 光标闪烁动画（1s 周期）
- 命令输出打字机效果（可选）
- 背景切换淡入淡出（0.5s 过渡）
- 页面加载时的欢迎信息逐行显示
